Endpoints

Base URL
https://api-spam-ui90.onrender.com
POST /check
Request body: { text: string }
Response: { label: 'spam'|'ham'|'likely_spam', score: number, reasons: string[] }

Usage Examples

# curl
curl -X POST 'https://api-spam-ui90.onrender.com/check' \
  -H 'content-type: application/json' \
  -H 'X-API-Key: <YOUR_KEY>' \
  -d '{"text":"Hello from curl"}'

// JavaScript (fetch)
async function checkSpam(text) {
  const res = await fetch('https://api-spam-ui90.onrender.com/check', {
    method: 'POST',
    headers: {
      'content-type': 'application/json',
      'X-API-Key': '<YOUR_KEY>'
    },
    body: JSON.stringify({ text })
  });
  return res.json();
}

# Python (requests)
import requests

url = 'https://api-spam-ui90.onrender.com/check'
headers = {
  'X-API-Key': '<YOUR_KEY>',
  'content-type': 'application/json'
}
payload = { 'text': 'Hello from Python' }
resp = requests.post(url, headers=headers, json=payload)
print(resp.status_code)
print(resp.json())

Error Codes

400 Bad Request
Invalid payload. Ensure body contains a non-empty "text" string.
401 Unauthorized
Missing or invalid RapidAPI key.
403 Forbidden
Key not subscribed to a plan or lacks access.
429 Too Many Requests
Rate limit exceeded. Respect per-minute limits by plan.
500 Internal Server Error
Unexpected error. Retry or contact support.
